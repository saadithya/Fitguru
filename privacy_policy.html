2. Information We Process

Information You Provide

  • Exercise names entered manually by the user
  • Optional API keys for third-party AI services (Google Gemini or OpenAI)

Media Accessed on Your Device

  • Workout videos recorded using the device camera
  • Images or videos selected by the user from device storage (if applicable)
  • Video frames extracted temporarily for AI-based exercise analysis

Information We Do NOT Collect

  • No personal identifiers (name, email, phone number)
  • No location data
  • No advertising identifiers
  • No analytics or tracking data

3. App Permissions

The App requires the following permissions to function:

Permission Purpose Data Handling
CAMERA Record workout videos for AI analysis Videos are stored locally on the device
RECORD_AUDIO Required by Android for video recording Audio is not analyzed, stored, or transmitted
READ_MEDIA_IMAGES Access images selected by the user for workout reference or analysis Images are processed locally and never uploaded
READ_MEDIA_VIDEO Access videos selected by the user for workout analysis or review Videos remain on-device and are not shared
Denying permissions may limit core functionality such as workout recording or analysis.

4. How Data Is Used

  • Analyze workout form using AI models
  • Provide feedback and insights to the user
  • Store workout history locally for progress tracking
We do not sell, share, rent, or monetize user data.

5. Third-Party AI Services

Users may optionally use their own API keys to connect with:

6. Data Storage & Security
